# Global FM - Interactive 3D Earth Globe with Radio Stations

An immersive 3D globe visualization application that displays a Google Earth-style Earth model with live radio stations from around the world as interactive pins. Built on the Internet Computer with React Three Fiber and Motoko.

![Global FM](./assets/astronaut-icon.png)

## Features

- **Interactive 3D Globe**: Ultra-realistic Earth visualization with 4K satellite textures
- **Live Radio Stations**: Thousands of radio stations from around the world with instant playback
- **Internet Identity Authentication**: Secure login with Internet Computer's Internet Identity
- **User Dashboard**: Track bookmarked channels and view user statistics
- **Space-Themed UI**: Immersive galaxy background with enhanced star field
- **Responsive Design**: Optimized for both desktop and mobile devices
- **Real-time Search**: Filter radio stations by name, country, or genre
- **Smooth Controls**: Ball-like physics for natural globe rotation
